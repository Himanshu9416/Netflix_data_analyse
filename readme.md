# Netflix Data Visualization 📊

##  Overview
An interactive Tableau dashboard that analyzes Netflix content by year, genre, country, and contributors.

##  Files in This Repository
| File                             | Description                                      |
|----------------------------------|--------------------------------------------------|
| `Netflix Analysis.twbx`          | Final Tableau Dashboard                          |

| `Netflix Project Proposal.pdf`   | Project Proposal Document                        |
| `README.md`                      | Project Overview and Guide                       |
| `netflix_data_cleaning.ipynb`     | Project                                         |

## 🚀 Dashboard Features
- Year-wise content trends
- Genre & country insights
- Top actors/directors
- Filters and interactivity

## 🛠 Tools Used
- Tableau
- GitHub

## 🎯 Outcome
An insightful dashboard to visualize Netflix's content growth, diversity, and contributor patterns.
